# Ad-blocker-extension-chrome
Ad blocker and remover for Google Chrome

Download Ad-blocker-extension-chrome.zip<br>
Unzip it<br>
Go to Chrome extensions: chrome://extensions/<br>
In the top right corner enable "Developer mode"<br>
In the top left corner click "Load unpacked"<br>
Select unzipped folder<br>
This extension will look for ads on all websites every 1000ms. You can adjust this time in the "content.js" file here: setInterval(removeGoogleAds, 1000);<br>
Enjoy!<br>
