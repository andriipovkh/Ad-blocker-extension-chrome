# Ad-blocker-extension-chrome
Ad blocker and remover for Google Chrome

Download Ad-blocker-extension-chrome.zip
Unzip it
Go to Chrome extensions: chrome://extensions/
In the top right corner enable "Developer mode"
In the top left corner click "Load unpacked"
Select unzipped folder
This extension will look for ads on all websites every 1000ms. You can adjust this time in the "content.js" file here: setInterval(removeGoogleAds, 1000);
Enjoy!
